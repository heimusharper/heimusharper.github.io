Исправление сломанного рельефа:

  • Нажмите сочетание клавиш Win+r
  • В отрывшемся окне "Run" введите в поле "Open": "%USERPROFILE%/appdata/local/fincor/supercam". Без кавычек, нажмите ОК
  • Откроется папка с насройками SUPERCAM для пользователя
  • Выберите свой профиль, скорее всего default
  • откройте через блокнот sesion.maps
  • Убрать содержимое "elevation_layer", оставить в следующем виде: ""elevation_layer": [],"
  • Сохранить, запустиь SUPERCAM, теперь можно загрузить свои файлы рельефа.
